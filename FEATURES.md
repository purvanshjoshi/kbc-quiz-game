# KBC Quiz Game Features & Enhancements

## Game Features

### Core Gameplay
- **Multiple Choice Questions**: 50+ carefully curated questions
- **Progressive Difficulty**: Questions get harder as you advance
- **Time Pressure**: 30-second timer per question
- **Lifelines**:
  - 50:50 - Removes two wrong answers
  - Phone a Friend - Shows AI suggestion
  - Audience Poll - Displays vote percentages

### Scoring System
- Tier-based rewards:
  - Tier 1 ($1,000)
  - Tier 2 ($5,000)
  - Tier 3 ($10,000)
  - Tier 4 ($25,000)
  - Tier 5 ($50,000)

### User Interface
- **Responsive Design**: Works on all devices
- **Dark Mode**: Easy on the eyes
- **Smooth Animations**: 60 FPS transitions
- **Accessibility**: WCAG 2.1 compliant
- **Sound Effects**: Custom audio feedback
